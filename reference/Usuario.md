# Usuario

Represents the "Usuario" structure in SEI (user info).

## Usage

``` r
Usuario(IdUsuario, Sigla, Nome)
```

## Arguments

- IdUsuario:

  Character. Identifier of the user.

- Sigla:

  Character. The user's sigla.

- Nome:

  Character. The user's full name.

## Value

An S3 object of class "Usuario".
