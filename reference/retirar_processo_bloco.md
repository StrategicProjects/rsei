# retirar_processo_bloco

Retira um processo de um bloco (`retirarProcessoBloco`).

## Usage

``` r
retirar_processo_bloco(
  id_bloco,
  protocolo_procedimento,
  config = sei_config(),
  verbose = FALSE
)
```

## Arguments

- id_bloco, protocolo_procedimento:

  Identificadores.

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
