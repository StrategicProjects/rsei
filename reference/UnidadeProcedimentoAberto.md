# UnidadeProcedimentoAberto

Represents the "UnidadeProcedimentoAberto" structure (where a process is
open).

## Usage

``` r
UnidadeProcedimentoAberto(Unidade, UsuarioAtribuicao = NULL)
```

## Arguments

- Unidade:

  A
  [`Unidade`](https://strategicprojects.github.io/rsei/reference/Unidade.md)
  object (the open unit).

- UsuarioAtribuicao:

  A
  [`Usuario`](https://strategicprojects.github.io/rsei/reference/Usuario.md)
  object to which the process is assigned (optional).

## Value

An S3 object of class "UnidadeProcedimentoAberto".
