# definir_marcador

Define marcadores em processos (`definirMarcador`).

## Usage

``` r
definir_marcador(definicoes, config = sei_config(), verbose = FALSE)
```

## Arguments

- definicoes:

  Lista de definições (cada uma
  `list(ProtocoloProcedimento=, IdMarcador=, Texto=)`).

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
