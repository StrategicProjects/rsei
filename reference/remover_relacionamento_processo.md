# remover_relacionamento_processo

Remove o relacionamento entre dois processos.

## Usage

``` r
remover_relacionamento_processo(
  protocolo1,
  protocolo2,
  config = sei_config(),
  verbose = FALSE
)
```

## Arguments

- protocolo1, protocolo2:

  Números dos processos.

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
