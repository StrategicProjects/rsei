# excluir_bloco

Exclui um bloco (`excluirBloco`).

## Usage

``` r
excluir_bloco(id_bloco, config = sei_config(), verbose = FALSE)
```

## Arguments

- id_bloco:

  Character. Número do bloco.

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
