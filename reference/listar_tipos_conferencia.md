# listar_tipos_conferencia

Lista os tipos de conferência.

## Usage

``` r
listar_tipos_conferencia(
  config = sei_config(),
  id_unidade = NULL,
  raw = FALSE,
  verbose = FALSE
)
```

## Arguments

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- id_unidade:

  Filtro opcional.

- raw, verbose:

  Logical.

## Value

Um `tibble` de tipos de conferência.
