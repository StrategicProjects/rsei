# atualizar_contatos

Atualiza (cadastra/altera) um conjunto de contatos
(`atualizarContatos`).

## Usage

``` r
atualizar_contatos(contatos, config = sei_config(), verbose = FALSE)
```

## Arguments

- contatos:

  Lista de contatos (objetos
  [`Contato()`](https://strategicprojects.github.io/rsei/reference/Contato.md)
  / listas nomeadas).

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
