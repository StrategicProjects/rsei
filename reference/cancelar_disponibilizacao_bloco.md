# cancelar_disponibilizacao_bloco

Cancela a disponibilização de um bloco
(`cancelarDisponibilizacaoBloco`).

## Usage

``` r
cancelar_disponibilizacao_bloco(
  id_bloco,
  config = sei_config(),
  verbose = FALSE
)
```

## Arguments

- id_bloco:

  Character. Número do bloco.

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
