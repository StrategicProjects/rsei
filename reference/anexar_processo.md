# anexar_processo

Anexa um processo a outro (`anexarProcesso`).

## Usage

``` r
anexar_processo(
  protocolo_principal,
  protocolo_anexado,
  config = sei_config(),
  verbose = FALSE
)
```

## Arguments

- protocolo_principal, protocolo_anexado:

  Números dos processos.

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
