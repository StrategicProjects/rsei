# listar_paises

Lista os países.

## Usage

``` r
listar_paises(config = sei_config(), raw = FALSE, verbose = FALSE)
```

## Arguments

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- raw, verbose:

  Logical.

## Value

Um `tibble` de países.
