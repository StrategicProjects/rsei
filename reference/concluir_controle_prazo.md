# concluir_controle_prazo

Conclui o controle de prazo de processos (`concluirControlePrazo`).

## Usage

``` r
concluir_controle_prazo(
  protocolos_procedimentos,
  config = sei_config(),
  verbose = FALSE
)
```

## Arguments

- protocolos_procedimentos:

  Vetor de números de processos.

- config:

  Um objeto
  [`sei_config()`](https://strategicprojects.github.io/rsei/reference/sei_config.md).

- verbose:

  Logical.

## Value

`TRUE` em caso de sucesso.
